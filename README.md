# 🖋️ Royal Journal - MERN Stack Note App

A sophisticated, full-stack note-taking application featuring a "Royal Blackcurrent" luxury aesthetic. Built with the MERN stack (MongoDB, Express, React, Node.js) and styled with Tailwind CSS.

## 🚀 Live Demo
[Check out the Live App here!](https://note-app-full-stack-seven.vercel.app/)

## ✨ Features
* **Elegant Design:** Custom high-end "Royal Blackcurrent" & Champagne Gold theme.
* **Full CRUD:** Create, Read, Update, and Delete notes seamlessly.
* **Organization:** Categorize entries (Work, Personal, Ideas) with dynamic tags.
* **Instant Search:** Real-time filtering by title or category.
* **Responsive:** Fully optimized for mobile and desktop viewing.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, Lucide Icons, Axios.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB Atlas.
- **Deployment:** Vercel (Frontend), Render (Backend).

## ⚙️ Installation & Setup
1. Clone the repo: `git clone https://github.com/Dhruvil7582/Note-App-Full-Stack.git`
2. Install dependencies for both folders:
   - `cd backend && npm install`
   - `cd frontend && npm install`
3. Add your `.env` file to the `/backend` folder with your `MONGO_URI`.
4. Start the development servers:
   - Backend: `npm start` (on port 5000)
   - Frontend: `npm run dev` (on port 5173)
